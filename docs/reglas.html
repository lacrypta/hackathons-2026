Reglas y Términos

Lightning Hackathons 2026 — La Crypta

1. Elegibilidad

  • Abierto a participantes de cualquier país
  • Menores de 18 años requieren consentimiento parental
  • Equipos de máximo 4 personas

2. Formato

  • Duración: 48 horas por hackathon
  • Modalidad: híbrido (presencial + remoto)
  • Entrega: código + demo + documentación

3. Código

  • Debe ser open source (MIT, Apache, GPL)
  • Publicado en GitHub público
  • No se permite código propietario
  • Librerías/frameworks externos permitidos

4. Evaluación

  • Jurado 100% AI (Claudio, Gorilatron, Gorilator)
  • Criterios: funcionalidad, utilidad, código, UX, innovación
  • Decisiones del jurado son finales
  • Plagio/fraude resulta en descalificación inmediata

5. Premios

  • Pagados en sats vía Lightning (LaWallet)
  • Ganadores notificados dentro de las 48hs post-hackathon
  • Wallet Lightning requerida para recibir premios

6. Propiedad Intelectual

  • El código es propiedad de los participantes
  • Licencia open source obligatoria
  • La Crypta puede integrar proyectos ganadores al stack
  • Atribución siempre requerida

7. Código de Conducta

  • Respeto, inclusión, colaboración
  • Cero tolerancia a acoso, discriminación, o abuso
  • Ver Código de Conducta completo
